## 适用场景

- 你有 Gemini API Key，希望快速使用云端模型翻译。

## 在 MinTranslate 里需要填写什么？

- **名称**
- **Model**
- **API Key（必填）**

可用于快速上手的模型示例：

- `gemini-2.5-flash`（项目默认值）

## 常见问题

- **提示 “Gemini API Key 未配置”**：去 `/settings` 给默认 Provider 填上 API Key。
- **AI 请求失败**：检查 Model 名称、Key、网络是否正常。

更多排障见：[常见问题（FAQ）](../faq)

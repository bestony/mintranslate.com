## 适用场景

- 你有 Anthropic 的 API Key，希望用 Claude 模型进行翻译。

## 在 MinTranslate 里需要填写什么？

- **名称**
- **Model**
- **API Key（必填）**

可用于快速上手的模型示例：

- `claude-3-5-haiku`（项目默认值）

## 常见问题

- **提示 “Anthropic API Key 未配置”**：去 `/settings` 给默认 Provider 填上 API Key。
- **AI 请求失败**：检查 Model 名称是否存在、网络是否正常、Key 是否有效。

更多排障见：[常见问题（FAQ）](../faq)

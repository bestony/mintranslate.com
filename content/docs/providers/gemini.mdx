## When to use

- You have a Gemini API key and want a simple cloud setup.

## Fields in MinTranslate

- **Name**
- **Model**
- **API Key** (required)

Example model names you can start with:

- `gemini-2.5-flash` (default in this project)

## Troubleshooting

- **“Gemini API key is not configured”**: add your key in `/settings`.
- **Request failed**: verify model name / key / network.

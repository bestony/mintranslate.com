## When to use

- You have an Anthropic API key and want to use Claude models for translation.

## Fields in MinTranslate

- **Name**
- **Model**
- **API Key** (required)

Example model names you can start with:

- `claude-3-5-haiku` (default in this project)

## Troubleshooting

- **“Anthropic API key is not configured”**: add your key in `/settings`.
- **Request failed**: double check the model name and your network.

See also: [FAQ](../faq)

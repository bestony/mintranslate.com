MinTranslate 是一个极简 AI 翻译器：你配置自己的模型 Provider（API Key 或本地 Ollama），MinTranslate 提供尽可能轻量、清晰的翻译体验。

## 30 秒快速开始

1. 打开[设置页](/settings)，新增一个模型供应商（OpenAI / Anthropic / Gemini / Ollama）。
2. 将它设为默认模型供应商。
3. 回到首页，在左侧输入文本，右侧会出现译文。

详细步骤见：[快速开始](./getting-started)

## 特性

- **BYOM（Bring your own model）**：你可以自由选择与管理 Provider（OpenAI / Anthropic / Gemini / Ollama）。
- **System prompt 可配置**：控制翻译风格、术语、格式保留等。
- **数据本地存储**：Provider 配置与 system prompt 保存在 localStorage；翻译历史保存在 IndexedDB，可随时导出/清空。

## 隐私与数据

- 翻译请求会从 **你的浏览器** 直接发送到你配置的模型供应商（MinTranslate 不代收、不转发、不保存你的翻译内容）。
- 模型供应商配置与 system prompt 会保存在你的浏览器 localStorage；翻译历史会保存在 IndexedDB（可在应用内清空，或在浏览器中清除站点数据）。
